Decision tree
How to run

1. Open dt_config file and provide the “input_file” param as the file path of input file
2. Run the script using python decision_tree.py


##################################################
Random Forest
How to run

1. Open rf_config file and provide the “input_file” param as the file path of input file
2. Provide the “k_fold_validation” as k-fold param
3. Provide the “features_for_split” as the feature for split param
4. Run the script using python random_forest.py
